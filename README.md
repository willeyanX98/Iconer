# Iconer 🚀

A simple, fast, and smart tool to convert images to `.ico` files.
Perfect for developers and designers who need custom icons for their projects.

## Features ✨

- **Single Conversion:** Convert any single image to `.ico` format.
- **Batch Conversion:** Convert all images in a folder with one click.
- **Format Support:** Supports PNG, JPG, JPEG, BMP, and WebP.
- **Smart Sizing:** Automatically handles aspect ratios and sizing.

## How to use 🛠️

1. Run `Iconer.exe`.
2. Choose your option (1 for single, 2 for batch).
3. Follow the on-screen instructions.

## Why Iconer? 🤔

- Lightweight and fast.
- No installation required.
- Perfect for custom Windows icons.
